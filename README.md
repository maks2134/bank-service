# Банковская система

Приложение для управления банковской базой данных с графическим интерфейсом на Go и Fyne.

## Требования

- Go 1.21+
- PostgreSQL 15+
- Docker и Docker Compose (для запуска БД)

## Установка и запуск

1. Запустите базу данных:
```bash
docker-compose up -d
```

2. Установите зависимости:
```bash
go mod tidy
```

3. Запустите приложение:
```bash
go run ./cmd/bank
```

## Функциональность

- Работа со всеми таблицами БД (CRUD операции)
- Управление таблицами (создание, удаление, редактирование структуры)
- Резервное копирование и восстановление
- Сохранение и выполнение SQL запросов
- Экспорт данных в Excel
- Предопределенные запросы для лабораторных работ 5 и 6

## Структура проекта

- `cmd/bank/` - точка входа приложения
- `internal/config/` - конфигурация
- `internal/models/` - модели данных
- `internal/repository/` - репозитории для работы с БД
- `internal/ui/` - графический интерфейс
- `internal/utils/` - утилиты (резервное копирование, экспорт, запросы)
- `pkg/db/` - подключение к БД
